# Portfólio de Antonio Miguel

Bem-vindo ao portfólio pessoal de **Antonio Miguel Borges Correa**. Este projeto apresenta um site estático em português que exibe informações sobre o autor, suas especialidades, trabalhos e formas de contato.

## Visão Geral

O site foi desenvolvido com HTML, CSS e JavaScript puro. Ele inclui:

- Navegação por seções: Sobre Mim, Especialidades, Trabalhos e Contato
- Menu responsivo para desktop e mobile
- Botão "voltar ao topo"
- Alternância de idioma com Google Translate (Português / Inglês)
- Galeria de trabalhos com pré-visualização em modal
- Links externos para GitHub, blog e LinkedIn

## Estrutura do Projeto

- `index.html` – página principal do portfólio
- `css/styles.css` – estilos principais do site
- `css/tamanho_mini_celular.css` – ajustes de layout para telas muito pequenas
- `css/tamanho_pc.css` – estilos adicionais para telas grandes
- `javascript/programs.js` – scripts de navegação, tradução e comportamento do botão de topo
- `javascript/scriptTrab.js` – galeria de imagens dos trabalhos com preview
- `img/` – imagens usadas no portfólio (ícones, fotos e gráficos)
- `trabalhos/` – imagens dos trabalhos exibidos na seção de portfólio
- `curriculo/` – pasta reservada para o currículo digital e arquivos relacionados

## Principais Recursos

- `MENU` flutuante que aparece ao rolar a página
- Seção **Sobre Mim** com resumo da trajetória e competências
- Seção **Especialidades** com:
  - Inglês
  - Desenvolvimento Desktop
  - Desenvolvimento Mobile
  - Desenvolvimento de Websites
  - Banco de Dados
  - Design Gráfico
- Seção **Trabalhos** com link para GitHub e blog, e galeria visual
- Seção **Contato** com telefone, email e LinkedIn

## Como Executar

1. Abra o arquivo `index.html` em um navegador moderno.
2. Nenhuma dependência adicional é necessária.
3. Para testar a tradução, clique na bandeira no canto superior.
